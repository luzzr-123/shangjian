## 1. 顶层导航

1. 今日
2. 任务
3. 习惯
4. 笔记

此外提供：
- 全局搜索入口
- 设置页
- 回收站页（可在设置内进入）
- 备份恢复页（可在设置内进入）

## 3. 关键流转
### 3.1 快速新建流
用户在任意主界面点击浮动按钮：
- 选择新建任务
- 选择新建习惯
- 选择新建笔记

进入对应创建页。

### 3.2 完成任务流
- 在今日页或任务页点击任务卡片
- 直接勾选子任务或整体完成
- 若规则为“全子任务完成自动完成”，则父任务自动完成

### 3.3 习惯打卡流
- 今日页展示今日应执行习惯
- 点击勾选打卡
- 步骤型或时长型习惯进入详情完成

### 3.4 笔记编辑流
- 新建或进入编辑页
- 输入 Markdown 内容
- 插入图片
- 自动保存草稿
- 返回时保留内容

# Milestones - 分阶段开发计划

## 总原则
- 每阶段都必须可运行
- 每阶段都必须可验收
- 不允许一口气把所有功能混在一起开发

## Milestone 0：项目初始化
目标：
- 建立一个能运行的安卓原生项目骨架。

范围：
- Compose
- Navigation
- Hilt
- Room
- WorkManager
- 主题系统
- 底部导航
- Today / Tasks / Habits / Notes 空页面

验收：
- 项目可编译运行
- 四个主页面可切换
- 包结构基本成型

## Milestone 1：任务模块 MVP
范围：
- 任务列表
- 新建任务
- 编辑任务
- 删除任务（软删除）
- 子任务
- 基础排序
- 截止时间显示
- 完成 / 恢复完成

验收：
- 可完整增删改查任务
- 子任务可增删改查
- 列表正确显示进度与状态

## Milestone 2：任务提醒系统
范围：
- 开始提醒
- 提醒窗口
- 特别提醒时间
- 重复提醒
- 完成后取消提醒
- 通知点击跳详情

验收：
- 至少在标准 Android 模拟器上可稳定触发提醒
- 完成任务后不再提醒
- 免打扰前先不做

## Milestone 3：习惯模块 MVP
范围：
- 习惯列表
- 新建习惯
- 编辑习惯
- 删除习惯
- 每日 / 每周 / 间隔 / 每月规则
- 勾选型 / 步骤型 / 时长型

验收：
- 今日应执行习惯可正确生成
- 可完成打卡
- 可撤销打卡

## Milestone 4：笔记模块 MVP
范围：
- 笔记列表
- 新建 / 编辑 / 删除笔记
- Markdown 存储
- Markdown 渲染
- 图片插入

验收：
- 可保存并展示 Markdown
- 可插入并渲染本地图片
- 返回后内容不丢失

## Milestone 5：今日页聚合
范围：
- 今日待办
- 今日习惯
- 最近笔记
- 快速新建
- 基础统计

验收：
- 打开应用即可看到今日关键信息
- 可直接进行常用操作

## Milestone 6：免打扰与设置
范围：
- 设置页
- 全局免打扰时间
- 通知权限提示
- 默认提醒参数

验收：
- 免打扰时间生效
- 设置项保存成功

## Milestone 7：回收站与备份恢复
范围：
- 回收站
- 手动恢复
- 彻底删除
- JSON + media 导出 zip
- zip 导入恢复

验收：
- 删除内容可找回
- 导出后可导入恢复

## Milestone 8：质量完善
范围：
- 单元测试
- ViewModel 测试
- Compose UI 测试
- 空状态优化
- 错误处理
- 小范围性能优化

验收：
- 核心流程测试覆盖
- 常见错误不崩溃
